# Experiments

Experiments are organized as Learning Units under `experiments/`.

Each Learning Unit should be restartable from repository artifacts without relying on prior chat history.

Default unit files:

- `README.md` — scope, concept, commands, expected observations, and simplifications.
- `STATUS.md` — current state, mode, next action, blockers, and resume instructions.
- `LEARNING.md` — Before, My Attempt, During, After.
- `HANDOFF.md` — optional, only for complex multi-session continuation.

Do not create a full spec for every lesson by default. Use a heavier spec only for complex design choices or multi-part plans.
